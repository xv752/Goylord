Downloading password-protected files:

Use the X-Download-Password header:

curl -H "X-Download-Password: yourpass" https://host/api/file-share/<id>/download -o file.ext

Or pass it as a query parameter:

curl "https://host/api/file-share/<id>/download?password=yourpass" -o file.ext
