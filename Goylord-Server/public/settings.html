Build IDs

Review your builds, see how many agents have claimed each, and block a build to immediately disconnect any agent that connects with its tag.

Claim count is the number of unique agent fingerprints that have ever connected using a build's tag. Each unique host produces one claim. A claim count higher than the number of machines you actually deployed to suggests the build tag may have leaked.
