Crypter Template

Template for building a crypter plugin. Replace the demo XOR with your own encryption.

How It Works

This plugin intercepts the build artifact via the onBuildArtifact hook and transforms the binary using the selected method:

  • XOR (Demo) — XORs each byte with the key. Working demo, not real encryption.
  • RC4 — Placeholder. Replace with your RC4 implementation.
  • AES-256 — Placeholder. Replace with your AES implementation + loader stub.

To Make a Real Crypter

  1. Replace the transform functions in server.js with your encryption logic
  2. Write a loader stub that decrypts at runtime and executes in memory
  3. Prepend the stub to the encrypted payload, or append it as a separate section
  4. Update the config.json settings to match your interface
